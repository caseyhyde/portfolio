THE PROBLEM

American Drapery Systems designs, builds, and installs custom commerical window treatments (blinds, shades, cubicle dividers, etc). The process for starting projects involves an intial site survey where measurements are taken of windows to be covered. Previously this was done in a note taking app on the client's tablet (Evernote™). The process was akin to writing in a blank notebook, to later be entered into spreadsheets, emails, and PDFs. It was repetetive, time consuming, and unorganized. Often, images were taken of the job site and relevant files were uploaded (office blueprints, etc), further adding to the difficulty of keeping everything in one place.

THE SOLUTION

Group Project

A comprhensive intial site survey, client information, and job management web application.

An all-in-one place to save customer information, create new site surveys, take measurements, and upload relevant files.

An organized central location to view and search through all current and past jobs, associated surveys, client information, and saved files.

PDF and Printer friendly export options, elimating the need to enter job information into secondary applications

THE PROBLEM

Skateboarders around the world enjoy skating at all sorts of unique spots. With the exception of established skateparks, none of these can be found via typical map services (Google Maps™). What about that sweet handrail behind the grocery store in uptown? How about the loading dock on the side of the downtown hardware store? How do skateboarders find these places if they are new to an area? How can they let the world know about a cool new spot they found tucked away in the suburbs?

THE SOLUTION

Spot Check! A web application developed to help skateboarders around the world share their favorite spots with other skateboards. New in town? Use the search feature to browse spots by location or keyword(stairs, ledge, hubba, rail, etc)... Found a sweet new spot that you think others should know about? Use the add spot feature to upload a new spot. Since a spot wouldn't be complete without photos, users can upload up to ten photos of their new spot at a time. The best part is, no registration required! Simply visit the website and begin browsing immidiately! To prevent robots from filling the spot database with nonsense, an email verification system is in place requiring users to verify their spot via email before it is hosted on the main website. The same system is used to allow only users who created a spot to edit/delete it.

WHAT IS IT?

The Game of Life, also known simply as Life, is a cellular automaton devised by the British mathematician John Horton Conway in 1970.
-Wikipedia

You saw this when you first came to my site. It is a zero-player game that uses a grid of "cells" (on this site 30x30, but could potentially be any size). There is an algorithm that checks each cell on the grid and it's neighbors state: alive or dead. Based on the amount of alive neighbors, each cell is toggled alive/dead or remains the same.

WHY DID I BUILD IT?

I love logic, and building Life is a wonderful practice in logical thinking involving nested for-loops with further nested switches. It is also a great match for Angualar's two-way data binding. This would typically be a job for two dimensional arrays, which Javascript doesn't offer, unlike other programming languages. As such, my sollution contains an array of arrays, which works wonderfully with Angular's ngRepeat.

I also added some extra features. When you toggle the "Methuselah Rain" feature, the server make a request to the Random.org API, which sends back an array of 10 "truly" random numbers between 0 and the number of cells on the board. I then use this array to place a Methuselah -- a pattern that takes a large number of generations in order to stabilize (known as its lifespan) and becomes much larger than its initial configuration at some point during its evolution. -Lifewiki -- onto the board at that corresponding position. The end result is a game that continuously grows, despite user interaction (making it a great thing to run in the backgrond and control stuff like the animations on this page)
